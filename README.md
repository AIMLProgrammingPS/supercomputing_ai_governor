# AI-Governed Stochastic Simulation Engine
An Operations Research framework for optimizing 10^7 vectorized simulation pipelines on resource-constrained hardware using a dynamic hardware-software co-design paradigm.

## Architecture
* `src/engine/`: High-performance Python 3.12 / NumPy simulation testbed.
* `src/governor/`: Real-time state-action controller monitoring hardware bottlenecks.
* `config/`: System thresholds and simulation parameter matrices.
